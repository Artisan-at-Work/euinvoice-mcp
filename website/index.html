Everything you need for EU invoice compliance

10 tools and 4 data resources covering the full European invoicing landscape.

%

VAT Rates

Standard, reduced, super-reduced, and parking rates for every country. Always current.

"What's the VAT rate for books in France?" → 5.5% (reduced rate)
⚡

E-Invoicing Requirements

B2G, B2B, and B2C mandates. Systems, platforms, timelines, and clearance models per country.

"Is e-invoicing mandatory in Italy?" → Yes — all transactions via SDI since 2019
◇

Format Standards

UBL, CII, Factur-X, XRechnung, FatturaPA, Peppol BIS — specs, pros/cons, and usage.

"What format for German B2G invoices?" → XRechnung 3.0 via ZRE/Peppol
✓

Field Validation

Required fields per jurisdiction and transaction type. Common EU rules plus country-specific extras.

"What fields for an invoice to Portugal?" → Standard fields + ATCUD, QR code, SAF-T hash
↔

Reverse Charge

When to apply reverse charge, what to put on the invoice, and how the buyer should handle it.

"Selling SaaS from NL to DE company" → Reverse charge Art. 196, no VAT
●

OSS Rules

One-Stop Shop schemes for cross-border B2C. Thresholds, registration, and reporting requirements.

"When do I need to register for OSS?" → Above €10,000 cross-border B2C

How it works

euinvoice is a Model Context Protocol server. It gives AI assistants direct access to structured EU invoicing data.

1

Install

One command. Works with any MCP-compatible client.

npm install -g euinvoice-mcp
2

Connect

Add to Claude Desktop, Cursor, or any MCP client config.

3

Ask

Your AI assistant now has complete EU invoicing knowledge. Ask in natural language.

31 countries covered

All EU-27 member states plus United Kingdom, Norway, Switzerland, and Iceland.

Data includes 2024/2025 rate changes and mandate updates.

Get started in 30 seconds

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "euinvoice": {
      "command": "npx",
      "args": ["-y", "euinvoice-mcp"]
    }
  }
}

Add to your MCP settings:

{
  "euinvoice": {
    "command": "npx",
    "args": ["-y", "euinvoice-mcp"]
  }
}
git clone https://github.com/Artisan-at-Work/euinvoice-mcp.git
cd euinvoice-mcp
npm install
npm run build
node dist/index.js
